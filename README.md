# Enhanced-Number-Guessing-Game
Built an **enhanced Number Guessing Game** in Python.

Description

Developed an enhanced Number Guessing Game in Python by extending a beginner-level project with additional gameplay features. The game includes multiple difficulty levels (Easy, Medium, and Hard), customizable number ranges, and different attempt limits for each level, making it more interactive and engaging.

Features

- Three difficulty levels: Easy, Medium, and Hard
- Customizable number range
- Different maximum attempts for each difficulty level
- Random number generation
- User input validation
- Feedback for correct and incorrect guesses
- Replay-friendly game logic

Technologies & Concepts

- Python
- "random" module
- Variables & Data Types
- User Input
- Conditional Statements ("if", "elif", "else")
- Loops ("while")
- Functions
- Exception Handling ("try-except")
- Input Validation
- Game Logic

Learning Outcomes

- Improved problem-solving and logical thinking
- Practiced writing modular and reusable Python code
- Strengthened understanding of loops, conditions, functions, and exception handling
- Learned to enhance a basic project by adding interactive features
